# HireLoop

HireLoop is a personal, single-user web application for managing autonomous job application workflows.

Current build status: **Phase 6 complete**. Google authentication, resume setup, encrypted LinkedIn credential storage, LinkedIn automation controls, AI fit scoring, company research, non-LinkedIn sourcing, conservative external form filling, Answer Memory, dashboard polish, Gmail outcome tracking, and deployment setup are implemented.
